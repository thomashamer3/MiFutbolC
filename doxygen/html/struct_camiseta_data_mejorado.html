Detailed Description

Estructura para almacenar datos de camiseta con análisis avanzado.

Esta estructura centraliza los datos de camiseta con análisis avanzado que se utilizan en todas las funciones de exportación, evitando la duplicación de código y facilitando el mantenimiento.
