Detailed Description

Declaraciones de funciones para importar datos en MiFutbolC.

Este archivo contiene las declaraciones de las funciones de importación que permiten cargar los datos del sistema desde archivos en diferentes formatos. Las funciones están organizadas por tipo de dato y formato de archivo.
