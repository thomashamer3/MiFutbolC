Detailed Description

Declaraciones de funciones utilitarias para entrada/salida y manejo de datos.

Este archivo de cabecera declara funciones auxiliares para interactuar con el usuario, manejar fechas y horas, limpiar la pantalla, verificar existencia de IDs en la base de datos, y gestionar directorios de exportación.

Ejecuta el comando apropiado para limpiar la pantalla dependiendo del sistema operativo. En Windows usa "cls", en sistemas Unix/Linux usa "clear".

◆ confirmar()

int confirmar ( const char * msg)

Solicita confirmación al usuario (Sí/No).

Muestra el mensaje proporcionado seguido de "(S/N):" y lee la respuesta del usuario. Acepta 's' o 'S' como afirmativo.

Parameters
msgEl mensaje a mostrar al usuario.
Returns
1 si el usuario confirma (sí), 0 en caso contrario.
