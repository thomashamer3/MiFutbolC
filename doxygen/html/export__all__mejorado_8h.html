void exportar_camisetas_todo_mejorado ( )

Exporta todos los datos de camisetas con análisis avanzado.

Exporta los datos de camisetas en todos los formatos disponibles incluyendo estadísticas avanzadas y análisis de rendimiento.

Exporta todos los datos de camisetas con análisis avanzado.

Esta función exporta los datos de camisetas en todos los formatos disponibles (CSV, TXT, JSON, HTML) con estadísticas avanzadas y análisis de rendimiento. Incluye métricas como eficiencia de goles, asistencias, porcentaje de victorias y análisis de lesiones por partido.

void exportar_todo_mejorado ( )

Exporta todos los datos con análisis avanzado.

Exporta todos los datos (camisetas y lesiones) en todos los formatos disponibles, incluyendo tanto los datos mejorados como los originales para compatibilidad.

Exporta todos los datos con análisis avanzado.

Esta función exporta todos los datos disponibles (camisetas y lesiones) en todos los formatos soportados (CSV, TXT, JSON, HTML), incluyendo tanto las versiones mejoradas con análisis avanzado como las versiones originales para mantener compatibilidad con versiones anteriores.

Muestra el menú para exportar datos mejorados.

Esta función muestra un menú interactivo que permite al usuario seleccionar qué tipo de datos mejorados desea exportar.

Muestra el menú para exportar datos mejorados.

Esta función muestra un menú interactivo que permite al usuario seleccionar entre diferentes opciones de exportación mejorada:

  • Exportar camisetas con análisis avanzado
  • Exportar lesiones con análisis de impacto
  • Exportar todos los datos con análisis avanzado
  • Volver al menú principal
